cCommandLine

cCommandLine: number
  • Defined in types.d.ts:1128
description

Requires to set the param string of the clientSendCommand function. The client will create a batch file RR_LocalData/command.bat. It executes the command and writes the output into RR_LocalData/_command.txt. If you want to keep your log output, you should redirect it into a file via your commandline. For example: myCommand 1>/fileserver/share/MyCommand.txt 2>/fileserver/share/MyCommand_error.txt myCommand 1>/fileserver/share/MyCommand.txt 2>&1 You can use all variables (like ) from render configs, as long as they are not job specific. The command buffer can only take up to 500 letters.
