_rrsStatsSaveDataLS

  • _rrsStatsSaveDataLS(): _rrsStatsSaveDataLS
    • Defined in types.d.ts:412
    description

    This class loads the file [RR]\sub\stats\server_stats.db This statistics file is used to create the server stats in rrControl, tab Global Info. There are three different stats: "day": A 24h statistic, saved every 90seconds. "month": A 20 day statistic, saved every 30 minutes. "year": A 160 days statistic, saved every 6 hours.
