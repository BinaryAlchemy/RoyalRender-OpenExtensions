
##########################################################################
### Description:

This client command sends a service restart command via a Windows tool on your machine to an other Windows machine to stop and then start the Royal Render service.




##########################################################################
### Installation:

Please copy the python script into [RR]\plugins\control_client\once\





##########################################################################
### Usage:

Select clients in rrControl, right-click and find this command in "Other >"
